This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

This project was built using an agentic development workflow, where AI agents were used to assist with planning, execution, and implementation rather than writing the code manually myself. The goal was to learn how modern agentic systems operate, including task decomposition, iterative planning, context management, decision-making, and execution workflows. Through this project, I gained experience in structuring requirements, guiding AI-driven development, reviewing generated code, debugging, and understanding how humans and AI can collaborate effectively in software engineering. The project serves as a learning experience in AI-assisted development methodologies and the practical application of agentic workflows.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
